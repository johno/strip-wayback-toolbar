Pixyll

Pixyll is a simple, beautiful theme for Jekyll that emphasizes content rather than aesthetic fluff. It’s mobile first, fluidly responsive, and delightfully lightweight. It’s pretty minimal, but leverages large type and drastic contrast to make a statement, on all devices.

Grayscale

Grayscale is a color palette for prototyping the web. It consists of three gray color series with Sass variables and corresponding CSS classes, allowing one to try out different grays while prototyping with the switch of a variable.

Urls for Humans

Urls for Humans is a gem that allows you to apply meaningful names to your Rails application's urls by leveraging what happens under the covers with the to_param and to_i methods. It's an extremely lightweight alternative to already existing solutions.

Ember Gravatar Addon

This Ember CLI Gravatar addon is an Ember component for gravatar image tags. It integrates with ember-cli, and has computed property binding, so it changes when the underlying email changes, as well.

Ember Dates Addon

This Ember CLI Dates addon has a few useful date helpers like time-ago-in-words, time-ahead-in-words, and time-delta-in-words that you can use in Ember applications.

Furtive CSS Micro-framework

Currently under development.

Furtive is a forward-thinking, lightweight CSS micro-framework. It's truly mobile-first. Meaning, all media queries are done by larger viewports. This cuts back on the processing necessary for mobile devices. It also has a small footprint, cutting down on the bandwidth necessary for downloading CSS. Furtive is intended to be just that, furtive. Minimal by design.

Lumber.js

Currently under development.

Lumber.js is a wrapper for the d3.js library. It comes with numerous graphs out of the box, and only requires you to write a touch of javascript. It leverages HTML5 data attributes to parse the graph's data and relevant information like color schemes and axis labels.

Bleuprint

Currently under development.

Yet another scaffold for static webpages. Bleuprint is intended to get you up and running quickly with Jekyll, Gulp build tasks, and responsive, mobile-first boilerplate.
